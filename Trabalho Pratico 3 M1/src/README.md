## Most Frequent Subtree Sum

De acordo com uma estrutura de árvore, será retornado a soma de cada nodo e seus filhos, se existirem. Os resultados mais frequentes serão parte da array de saída.

Caso 1:\
root = [5,2,-3]\
output = 2 -3 4

Caso 2:\
root = [5,2,-5]\
output = 2

Caso 3:\
root = [5,14,null,1]\
output = 1 15 20
